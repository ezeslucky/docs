description: A guide to using the Deployi command-line interface
---

Deployi CLI is a command-line tool for remotely managing your Deployi server. It simplifies creating, deploying, and managing applications and databases.

## Installation

```bash
npm install -g @deployi/cli
```

## Usage

```bash
deployi COMMAND
```
To get help on a specific command:

```bash
deployi COMMAND --help
```
