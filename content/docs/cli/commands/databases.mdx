The Deployi CLI can be used to create, deploy, and manage databases.

## Requirements

Is required to be already authenticated with the Deployi CLI.


## Commands

### MariaDB 

1. `deployi database mariadb create` - Create a new mariadb database.
2. `deployi database mariadb delete` - Delete an mariadb database.
3. `deployi database mariadb deploy` - Deploy a mariadb database.
4. `deployi database mariadb stop` - Stop a running mariadb database.
