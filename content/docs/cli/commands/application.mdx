The Deployi CLI can be used to create, deploy, and manage applications.

## Requirements

Is required to be already authenticated with the Deployi CLI.


## Commands

1. `deployi app create` - Create a new application.
2. `deployi app delete` - Delete an application.
3. `deployi app deploy` - Deploy an application.
4. `deployi app stop` - Stop a running application.
