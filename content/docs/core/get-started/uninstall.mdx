title: Uninstall
description: "Learn how to uninstall Deployi on your server"
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

Follow these steps to completely remove deployi and its components from your server.

<Steps>
<Step>
Remove the docker swarm services created by deployi:

   ```bash
   docker service rm deployi deployi-traefik deployi-postgres deployi-redis
   ```
 
</Step>
 
<Step>
 
Remove the docker volumes created by deployi:

   ```bash
   docker volume rm -f deployi-postgres-database redis-data-volume
   ```
 
</Step>
 
<Step>

Remove the deployi files and directories from your server:

   ```bash
   sudo rm -rf /etc/deployi
   ```
 
</Step>
</Steps>
