## 卸载步骤

按照以下步骤从您的服务器中完全删除 deployi 及其组件。

### 第 1 步： 删除 Docker Swarm 服务

首先，删除 Docker Swarm 中的 deployi 以及相关服务:

```bash
docker service rm deployi deployi-traefik deployi-postgres deployi-redis
```

### 第 2 步： 删除 Docker 挂载卷

接下来，删除 deployi 创建的 Docker 卷：

```bash
docker volume rm -f deployi-postgres-database redis-data-volume
```

### 第 3 步： 删除 deployi 文件

最后，从服务器上删除 deployi 文件和目录：

```bash
sudo rm -rf /etc/deployi
```
